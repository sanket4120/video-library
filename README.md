## How to install and run locally ?

```
$ git clone https://github.com/sanket4120/video-library.git
$ cd video-library
$ npm install
$ npm start
```
## Features -

- Home page
- Video listing page
- Filter Videos
- Search video
- Individual video page
- Like video 
- Watch Later Video
- History video
- Create playlist
- Authentication:
  - User Signup
  - User Login
 - Fully Responsive

## Built with -
- ReactJS
- [Modulo](https://modulo.netlify.app/) component library
- [Mockbee](https://mockbee.netlify.app/) for backend
- React Router
- React Context API
